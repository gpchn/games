🎨 科艺节 · AI创意方案册

基于文本大模型 · 零硬件 · 可部署于 Vercel / 静态托管

📁 第一版方案 · 通用灵感

✍️ 纯文字组

6个项目
  • 1 AI对联生成器
    简介：用户输入上联，AI生成下联，可调整风格（工整/趣味）。
    前端HTML/CSS/JS + Vercel Python函数 + 文本API（Qwen/GLM）
  • 2 智能谜语猜猜看
    简介：AI随机出谜语，用户输入答案，AI判断对错并给出提示。
    前端静态页面 + Vercel Python函数 + 文本API + 本地缓存谜语列表
  • 3 故事接龙
    简介：多人轮流点击“续写”，AI根据前文生成下一段故事，故事逐步累积。
    前端显示故事 + Vercel Python函数维护上下文 + 文本API
  • 4 观点辩论生成器
    简介：用户输入话题，AI同时生成正反方论点，并随机分配“支持率”柱状图。
    前端分栏显示 + Chart.js简单柱状图 + Vercel Python函数 + 文本API
  • 5 诗词创作助手
    简介：输入主题（如“毕业”“春天”），AI生成一首古典或现代诗，可切换风格。
    前端下拉菜单选择风格 + Vercel Python函数 + 文本API
  • 6 校园新闻标题生成器
    简介：选择几个关键词（如“运动会”“夺冠”），AI生成夸张有趣的新闻标题。
    前端卡片式UI + Vercel Python函数 + 文本API

🎬 视觉组

7个项目
  • 1 情绪调色盘
    简介：用户输入任何文本，AI分析情感和主要意象，屏幕背景渐变色随情感变化，飘落与意象相关的粒子（如“海”飘水滴）。
    前端p5.js（粒子系统） + Vercel Python函数 + 文本API
  • 2 动态词云
    简介：用户粘贴文章或输入短句，AI提取关键词并赋情感值，生成浮动旋转的词云，颜色代表情感，大小代表权重。
    前端wordcloud2.js + Vercel Python函数 + 文本API
  • 3 故事场景图标
    简介：AI生成微型故事并返回表情符号序列（如🏔️🌊🏠），前端依次高亮显示对应图标，营造场景感。
    前端FontAwesome图标 + 动画高亮 + Vercel Python函数 + 文本API
  • 4 墨迹诗句
    简介：AI生成一句古风诗句，屏幕以毛笔字体逐字显示，背景伴随墨滴扩散的粒子效果。
    前端Canvas（墨迹粒子模拟） + Google Fonts毛笔字体 + Vercel Python函数 + 文本API
  • 5 辩论分屏
    简介：左右分屏显示AI生成的正反方观点，背景分别用冷暖色，底部显示简易柱状图表示论点“强度”。
    前端CSS flex布局 + Chart.js + Vercel Python函数 + 文本API
  • 6 谜语烟花
    简介：AI出谜语，用户猜对后屏幕绽放烟花粒子；猜错显示鼓励文字。
    前端Canvas烟花效果 + Vercel Python函数 + 文本API
  • 7 新闻头条动画
    简介：AI生成的校园新闻标题以老式报纸头条风格显示，伴有纸张纹理和打字机音效（可选）。
    前端CSS动画 + 音频（可选） + Vercel Python函数 + 文本API

🇨🇳 视觉组 · 红色传承

7个项目
  • 1 红心向党粒子墙
    简介：用户点击“我向党表白”，屏幕出现红色粒子汇聚成党徽或红心，并显示一句AI生成的青春宣言。
    前端Canvas粒子系统 + Vercel Python函数 + 文本API
  • 2 青春词云·红色版
    简介：用户输入一段爱国感言，AI提取关键词生成词云，红色系配色，词形可排成党徽或五角星轮廓。
    前端wordcloud2.js + Vercel Python函数 + 文本API
  • 3 红色地标灯光秀
    简介：屏幕显示简笔画版红色地标（如延安宝塔、天安门），用户点击“点亮”，AI生成一句相关历史介绍，同时地标被“点亮”发光。
    前端SVG/Canvas动画 + Vercel Python函数 + 文本API
  • 4 青春誓言书法屏
    简介：AI生成的青春誓言以毛笔字体逐字写出，背景有红色飘带粒子效果。
    前端Canvas毛笔字动画 + 粒子效果 + Vercel Python函数 + 文本API
  • 5 党史大事记时间轴
    简介：用户滑动时间轴（1921、1949、1978等），AI生成该年份的关键事件简介，同时背景颜色和粒子效果随年代变化。
    前端滑动条 + 时间轴UI + Vercel Python函数 + 文本API
  • 6 红色电影台词生成
    简介：选择经典红色电影（如《建党伟业》），AI生成一句仿风格的台词，并配上动态胶片背景。
    前端胶片滚动动画 + Vercel Python函数 + 文本API
  • 7 我和我的祖国·关键词作画
    简介：用户输入“青春”“奉献”等词，AI生成一句祝福语，屏幕飘落红色五角星粒子，排列成词形。
    前端粒子系统 + Vercel Python函数 + 文本API
⚡ 所有方案均使用纯前端 + Vercel Python无服务器函数，可打包为Electron应用离线运行 · 文本API使用你已有的Qwen/GLM/DeepSeek系列
